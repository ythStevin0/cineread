# 🎬 CineRead

Premium movie & book catalog platform with official trailers and synopses.

## Project Structure

| Folder   | Tech Stack                  | Hosting  |
|----------|-----------------------------|----------|
| `/backend` | Node.js + Express + MongoDB | Railway  |
| `/frontend`| React + Vite + Tailwind CSS | Vercel   |

## ✨ Features

- **Movie & TV Discovery**: Trending & popular movies/series from TMDB API with official YouTube trailers
- **Book Catalog**: Featured & popular books from Google Books API  
- **User System**: Authentication, favorites, viewing history
- **Responsive Design**: Premium dark UI/UX with glassmorphism & smooth animations
- **Search**: Unified search across movies, TV shows & books
- **Modal Previews**: Immersive full-screen trailers & book details

## 🛠 Tech Stack

**Backend**: Node.js, Express, MongoDB, JWT Auth  
**Frontend**: React 18, Vite, Tailwind CSS 3, React Router, Zustand  
**APIs**: TMDB (movies/TV), Google Books, YouTube  
**Deployment**: Railway (backend), Vercel/Netlify (frontend)

## 🚀 Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend  
cd frontend && npm install && npm run dev
```
